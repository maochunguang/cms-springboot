# wxcms系统 springboot版本
## 特性
1. 多数据源拆分
2. 多环境配置，dev，test，prod三个环境
3. redis集群配置
4. 日志分类
5. 外网映射（方便调试）
6. 微信公众号支持
7. 微信公众号运营基本功能（菜单，打标签）

## 环境
jdk1.8
tomcat8.0
mysql5.7
maven构建
jenkins发布
